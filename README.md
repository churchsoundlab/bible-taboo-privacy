# Bible Taboo — Privacy Policy site

Static one-page privacy policy for the **Bible Taboo** Android app, served via GitHub Pages.

- **Live URL:** https://churchsoundlab.github.io/bible-taboo-privacy/
- **Public repo:** https://github.com/churchsoundlab/bible-taboo-privacy
- Hosting: GitHub Pages, source = `main` branch, root (`/`), serving `index.html`.

This folder lives inside the app project (`bible-game-taboo/privacy/`) but is its **own git
repo** pushed to the separate **public** `bible-taboo-privacy`. It is git-ignored by the
private app repo (see the app's `.gitignore`) so the two never mix. Editing the policy: change
`index.html` here, commit, and push — Pages redeploys automatically. Keep the wording in sync
with the app's `PRIVACY.md`.
